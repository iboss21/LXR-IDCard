# Assets Directory

This directory contains visual and audio assets for The Land of Wolves ID Card System.

## Required Assets

### Images

1. **background.png** (optional)
   - Aged parchment texture background
   - Dimensions: 1920x1080px
   - Use: Card background overlay

2. **stamp.png** (optional)
   - Red "APPROVED" citizenship stamp
   - Format: PNG with transparency
   - Dimensions: 800x800px
   - Use: Overlay on approved citizenship cards

3. **frame.png** (optional)
   - Ornate photo frame with wolf engravings
   - Format: PNG with transparency
   - Use: Photo border on ID cards

4. **wolf_emblem.png** (optional)
   - Wolf head or howling wolf silhouette
   - Format: PNG with transparency
   - Use: Decorative emblem

### Audio (Optional)

1. **paper_rustle.ogg**
   - Paper shuffling sound effect
   - Use: When opening ID card

2. **stamp_sound.ogg**
   - Rubber stamp sound
   - Use: When stamp appears on approved cards

### Inventory Item Images

These should be placed in your inventory's image directory:

1. **photo_plate.png** - Camera or photograph icon (128x128px)
2. **id_pending.png** - Document without stamp (128x128px)
3. **id_citizen.png** - Document with official seal (128x128px)

## Notes

- All images are optional - the UI works with CSS styling alone
- If you add custom images, update the CSS to reference them
- Keep file sizes reasonable for faster loading (<500KB per image)
- Audio files should be OGG format for RedM compatibility
